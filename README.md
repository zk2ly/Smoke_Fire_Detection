# Smoke_Fire_Detection

烟雾 火焰检测

## 介绍

用的yolov5框架

weights中smoke.pt是用torch1.4以上版本用的，smoke2.pt是torch1.4及其以下版本用的。

如需准对特定场景数据集进行Fine-Tune,请参考yolov5项目: https://github.com/ultralytics/yolov5
